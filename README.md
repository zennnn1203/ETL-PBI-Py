# ETL Sales Data Analysis

## Project Overview
This project implements an **end-to-end ETL pipeline** for sales data using **Python and Power BI**. The goal is to extract, transform, and visualize sales data to generate actionable business insights.

## Features
- **Extract** data from Excel/CSV files containing sales transactions.
- **Transform** data using Python libraries:
  - `pandas` for data manipulation
  - `numpy` for numerical operations
  - Data cleaning, missing value handling, type conversion
- **Explore** data using:
  - `seaborn` and `matplotlib` for visual analysis
  - Identify trends, top-performing products and regions
- **Load & Visualize** data in Power BI:
  - Interactive dashboards
  - Analysis of sales, profit, product categories, regions
  - Insights on discounts, shipping costs, and profitability trends

## Dataset
The dataset is based on **SuperStore US transactions**, containing fields such as:
- Order ID, Customer ID
- Product Category, Sub-Category
- Sales, Profit, Discount
- Region, State, City
- Ship Mode, Segment

> Source: [Kaggle - Store Transactions Full Analysis](https://www.kaggle.com/code/amrboghdady74/store-transactions-full-analysis/notebook)

## Tools & Libraries
- **Python**: `pandas`, `numpy`, `matplotlib`, `seaborn`
- **Power BI**: Interactive dashboards and visualizations
- **Excel/CSV**: Original raw dataset
